Go 安装指南

请根据您计算机的操作系统选择下方的选项卡，然后按照对应的安装说明进行操作。

  1. 移除任何先前的 Go 安装：删除 /usr/local/go 文件夹（如果存在），然后将下载的归档文件解压到 /usr/local，在 /usr/local/go 创建一个新的 Go 目录树： 
          $ rm -rf /usr/local/go && tar -C /usr/local -xzf go1.14.3.linux-amd64.tar.gz

    (您可能需要以 root 用户身份或通过 sudo 分别运行每条命令。)

    不要将归档文件解压到现有的 /usr/local/go 树中。已知这样会导致 Go 安装不完整。

  2. 将 /usr/local/go/bin 添加到 PATH 环境变量。

    您可以通过在 $HOME/.profile 或 /etc/profile（用于系统范围的安装）中添加以下行来实现： 

              export PATH=$PATH:/usr/local/go/bin

    注意：对配置文件的更改可能要到您下次登录计算机时才会生效。要立即应用更改，只需直接运行 shell 命令，或使用类似 source $HOME/.profile 的命令从配置文件执行它们。

  3. 将 $HOME/go/bin（默认的 Go bin 路径）添加到您的 PATH 环境变量，以便使用通过 go install 安装的工具。

    您可以通过在 $HOME/.profile 或 $HOME/.bashrc 文件中添加以下行来实现： 

              export PATH="$PATH:$(go env GOPATH)/bin"
  4. 重启所有已打开的终端会话以使更改生效。

  5. 打开命令提示符并键入以下命令以验证您是否已安装 Go： 
              $ go version
  6. 确认该命令输出您已安装的 Go 版本。
  1. 打开您下载的包文件并按照提示安装 Go。

    该包会将 Go 分发版安装到 /usr/local/go。该包通过创建 /etc/paths.d/go 文件，将 /usr/local/go/bin 目录添加到您的 PATH 环境变量。

  2. 将 $HOME/go/bin（默认的 Go bin 路径）添加到您的 PATH 环境变量，以便使用通过 go install 安装的工具。

    您可以通过在 ~/.zshrc 或 ~/.bash_profile 文件中添加以下行来实现： 

              export PATH="$PATH:$(go env GOPATH)/bin"
  3. 重启所有已打开的终端会话以使更改生效。

  4. 打开一个新的终端会话并键入以下命令以验证您是否已安装 Go： 
              $ go version
  5. 确认该命令输出您已安装的 Go 版本。
  1. 打开您下载的 MSI 文件并按照提示安装 Go。

    默认情况下，安装程序会将 Go 安装到 Program Files 或 Program Files (x86)。您可以根据需要更改位置。安装完成后，您需要关闭并重新打开任何已打开的命令提示符，以便安装程序对环境所做的更改能在命令提示符中生效。

  2. 验证您是否已安装 Go。
    1. 在 Windows 中，点击 开始 菜单。
    2. 在菜单的搜索框中，键入 cmd，然后按 Enter 键。
    3. 在出现的命令提示符窗口中，键入以下命令： 
                    $ go version
    4. 确认该命令输出您已安装的 Go 版本。
