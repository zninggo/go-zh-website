本页汇总了Go语言官方稳定版本之间的变更内容。 完整的详细信息请查阅变更日志。

如需更新到特定版本，请使用以下命令：

git fetch --tags
git checkout goX.Y.Z

发布策略

每个Go语言主要版本将获得支持，直到后续出现两个新的主要版本。 例如，Go 1.5在Go 1.7发布前一直获得支持，Go 1.6则在Go 1.8发布前持续获得支持。 我们会根据需要通过发布次要修订版本（例如Go 1.6.1、Go 1.6.2等）来修复支持版本中的关键问题，包括关键安全问题。

go1.8（发布于 2017年02月16日）

Go 1.8是Go语言的一个主要版本。 详情请阅读Go 1.8 发布说明。

次要修订

go1.8.1（发布于 2017年04月07日）包含了对编译器、链接器、运行时、文档、go 命令以及 crypto/tls、encoding/xml、image/png、net、net/http、reflect、text/template 和 time 包的修复。 详情请查阅我们的问题跟踪器中的 Go 1.8.1 里程碑。

go1.8.2（发布于 2017年05月23日）包含了针对 crypto/elliptic 包的安全修复。 详情请查阅我们的问题跟踪器中的 Go 1.8.2 里程碑。

go1.8.3（发布于 2017年05月24日）包含了对编译器、运行时、文档以及 database/sql 包的修复。 详情请查阅我们的问题跟踪器中的 Go 1.8.3 里程碑。

go1.8.4（发布于 2017年10月04日）包含了两个安全修复。 它包含了与Go 1.9.1相同的修复，并在同一时间发布。 详情请查阅我们的问题跟踪器中的 Go 1.8.4 里程碑。

go1.8.5（发布于 2017年10月25日）包含了对编译器、链接器、运行时、文档、go 命令以及 crypto/x509 和 net/smtp 包的修复。 它修复了一个在Go 1.8.4中引入的错误，该错误导致在某些条件下 go get 无法获取非Git仓库。 详情请查阅我们的问题跟踪器中的 Go 1.8.5 里程碑。

go1.8.6（发布于 2018年01月22日）包含了与Go 1.9.3中 math/big 包相同的修复，并在同一时间发布。 详情请查阅我们的问题跟踪器中的 Go 1.8.6 里程碑。

go1.8.7（发布于 2018年02月07日）包含了针对“go get”的安全修复。 它包含了与Go 1.9.4相同的修复，并在同一时间发布。 详情请查阅我们的问题跟踪器中的 Go 1.8.7 里程碑。

go1（发布于 2012-03-28）

Go 1 是 Go 语言的一个重要版本，将保持长期稳定。 请阅读 Go 1 版本说明 了解更多信息。

为 Go 1 编写的程序预计将继续在 Go 1 的未来版本中无需更改即可正确编译和运行。 关于 Go 1 的未来，请阅读 Go 1 兼容性文档 了解更多。

go1 发布版本对应 weekly.2012-03-27。

次要修订版

go1.0.1（发布于 2012-04-25）是为了修复一个可能导致内存损坏的逃逸分析错误而发布的。 它还包含一些次要的代码和文档修复。

go1.0.2（发布于 2012-06-13）是为了修复在使用结构体或数组作为键的 map 实现中的两个错误：issue 3695 和 issue 3573。 它还包含许多次要的代码和文档修复。

go1.0.3（发布于 2012-09-21）包含次要的代码和文档修复。

完整变更列表请查阅 go1 发布分支历史。
