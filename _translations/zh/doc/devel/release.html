本页总结了 Go 官方稳定版本之间的变更。 详细的完整信息请参阅 变更日志。

要更新到特定版本，请使用：

git fetch --tags
git checkout goX.Y.Z

发布策略

每个主要的 Go 版本都会获得支持，直到有两个更新的主要版本发布。 例如，Go 1.5 的支持持续到 Go 1.7 发布，而 Go 1.6 的支持持续到 Go 1.8 发布。 我们会根据需要，在受支持的版本中修复关键问题（包括严重的安全问题）， 方法是发布次要修订版（例如，Go 1.6.1、Go 1.6.2 等）。

go1.8 (发布于 2017-02-16)

Go 1.8 是 Go 的一个主要版本。 欲了解更多信息，请阅读 Go 1.8 发行说明。

次要修订版

go1.8.1 (发布于 2017-04-07) 包含对编译器、链接器、运行时、文档、go 命令以及 crypto/tls、encoding/xml、image/png、net、net/http、reflect、text/template 和 time 包的修复。 详情请查阅我们问题跟踪器上的 Go 1.8.1 里程碑。

go1.8.2 (发布于 2017-05-23) 包含对 crypto/elliptic 包的安全修复。 详情请查阅我们问题跟踪器上的 Go 1.8.2 里程碑。

go1.8.3 (发布于 2017-05-24) 包含对编译器、运行时、文档以及 database/sql 包的修复。 详情请查阅我们问题跟踪器上的 Go 1.8.3 里程碑。

go1.8.4 (发布于 2017-10-04) 包含两个安全修复。 它包含了与 Go 1.9.1 相同的修复，并且是同时发布的。 详情请查阅我们问题跟踪器上的 Go 1.8.4 里程碑。

go1.8.5 (发布于 2017-10-25) 包含对编译器、链接器、运行时、文档、go 命令以及 crypto/x509 和 net/smtp 包的修复。 它修复了在 Go 1.8.4 中引入的一个错误，该错误导致在某些条件下对非 Git 仓库执行 go get 时出错。 详情请查阅我们问题跟踪器上的 Go 1.8.5 里程碑。

go1.8.6 (发布于 2018-01-22) 包含了与 Go 1.9.3 相同的 math/big 修复，并且是同时发布的。 详情请查阅我们问题跟踪器上的 Go 1.8.6 里程碑。

go1.8.7 (发布于 2018-02-07) 包含对 "go get" 的安全修复。 它包含了与 Go 1.9.4 相同的修复，并且是同时发布的。 详情请查阅我们问题跟踪器上的 Go 1.8.7 里程碑。

go1.7 (发布于 2016-08-15)

Go 1.7 是 Go 的一个主要版本。 欲了解更多信息，请阅读 Go 1.7 发行说明。

次要修订版

go1.7.1 (发布于 2016-09-07) 包含对编译器、运行时、文档以及 compress/flate、hash/crc32、io、net、net/http、path/filepath、reflect 和 syscall 包的修复。 详情请查阅我们问题跟踪器上的 Go 1.7.1 里程碑。

go1.7.2 不应使用。它已被标记，但并未完全发布。 由于最后一刻收到的错误报告，该版本的发布被推迟。 请使用 go1.7.3，并参考下面的变更摘要。

go1.7.3（发布于 2016 年 10 月 19 日）包含对编译器、运行时以及 crypto/cipher、crypto/tls、net/http 和 strings 包的修复。 详情请查阅我们问题跟踪器上的 Go 1.7.3 里程碑。

go1.7.4（发布于 2016 年 12 月 1 日）包含两项安全修复。 详情请查阅我们问题跟踪器上的 Go 1.7.4 里程碑。

go1.7.5（发布于 2017 年 1 月 26 日）包含对编译器、运行时以及 crypto/x509 和 time 包的修复。 详情请查阅我们问题跟踪器上的 Go 1.7.5 里程碑。

go1.7.6（发布于 2017 年 5 月 23 日）包含与 Go 1.8.2 相同的安全修复，并在同一时间发布。 详情请查阅我们问题跟踪器上的 Go 1.8.2 里程碑。

go1（发布于 2012 年 3 月 28 日）

Go 1 是 Go 的一个重要版本，将长期保持稳定。 更多信息请阅读 Go 1 发布说明。

旨在为 Go 1 编写的程序将继续在未来的 Go 1 版本下正确编译和运行，无需更改。 有关 Go 1 未来的更多信息，请阅读 Go 1 兼容性文档。

go1.0.3（发布于 2012 年 9 月 21 日）包含次要的代码和文档修复。
