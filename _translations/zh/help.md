<section id="help" class="BigCard">
<h2>获取帮助</h2>
<h3 id="mailinglist"><a href="https://groups.google.com/group/golang-nuts" aria-describedby="help-description">Go Nuts 邮件列表</a></h3>
<p>
在官方邮件列表中从 Go 用户那里获取帮助，并分享你的工作。
</p>
<p>
在发帖之前，请搜索 <a href="https://groups.google.com/group/golang-nuts" aria-describedby="help-description">golang-nuts</a>
的存档，并参考 <a href="/doc/go_faq.html">常见问题解答</a> 和
<a href="/wiki">维基</a>。
</p>

<h3 id="forum"><a href="https://forum.golangbridge.org/" aria-describedby="help-description">Go 论坛</a></h3>
<p>
<a href="https://forum.golangbridge.org/" aria-describedby="help-description">Go 论坛</a> 是一个面向 Go 程序员的讨论论坛。
</p>

<h3 id="discord"><a href="https://discord.gg/golang" aria-describedby="help-description">Gophers Discord</a></h3>
<p>
在 Go Discord 上获得实时支持并与其他 gophers 交流。
</p>

<h3 id="slack"><a href="https://blog.gopheracademy.com/gophers-slack-community/" aria-describedby="help-description">Gopher Slack</a></h3>
<p>在 Go Slack 频道中从其他用户那里获得实时支持。</p>

<h3 id="irc"><a href="ircs:irc.libera.chat/go-nuts" aria-describedby="help-description">Go IRC 频道</a></h3>
<p>在 Go IRC 频道 <code>irc.libera.chat</code> 上的 <code>#go-nuts</code> 获取实时支持。</p>

<h3 id="faq"><a href="/doc/faq" aria-describedby="help-description">常见问题解答 (FAQ)</a></h3>
<p>关于 Go 的常见问题解答。</p>
</section>
